WeatherWear — Personalized Recommendation Engine

Prototype

Personalized clothing recommendation engine built in a single React file. Six normalized environmental signals, dynamic feature weighting with a 30% bias cap, explainable scoring on every recommendation, and a live feedback loop that tracks precision/recall/F1 and exports structured training data as JSON. Works offline — no API key, no backend.

  • Feature weighting engine with iterative redistribution — no single signal dominates
  • Explainability built in: score breakdown, color-coded contribution bar, top driving factor per item
  • Feedback system exports labeled training records — architectured for model iteration from day one

Royals Player Insights Dashboard

Complete

Local analytics engine for KC Royals player analysis. One CLI command returns SARIMAX forecasting, feature importance rankings, partial dependence plots, causal inference estimates, and anomaly detection. Synthetic data fallback makes it CI-friendly. Goes past "what happened" to "what drives performance."

  • Causal inference via DoWhy/EconML — separates correlation from causation in performance data
  • Graceful degradation: EWMA fallback, no-Streamlit mode, synthetic data — works in any environment
  • CLI-first design: automatable, scriptable, no UI dependency
